# Cat Dots

Dotfiles _*almost*_ based on Catppuccin Mocha colorscheme.

## Installation
### Package Manager

```bash
# MacOS or Linux
brew install stow

# Arch (BTW)
pacman -S stow

#Fedora
dnf install stow

#Debian
apt install stow
```

### Clone this repo
```git clone https://github.com/F4LC0N69/dotfiles.git && cd dotfiles```


### Simply stow the files you need or copy them if you want to. Eg:-

```bash
#For zsh config
stow zsh


```
